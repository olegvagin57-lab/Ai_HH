# 🎯 План функционального тестирования HH Resume Analyzer
## Основной функционал системы подбора персонала

### 📋 Основная цель системы:
**Помочь HR-специалистам найти наиболее подходящие резюме с HeadHunter для конкретных вакансий с использованием ИИ-анализа**

---

## 🔄 Основной пользовательский сценарий

### Сценарий: "Подбор кандидатов на вакансию"

#### Шаг 1: Авторизация
- Пользователь заходит на сайт
- Вводит логин/пароль
- Система проверяет данные и авторизует

#### Шаг 2: Создание поиска
- Пользователь указывает город (например: "Москва")
- Описывает требования к кандидату (например: "Python разработчик с опытом Django")
- Запускает поиск

#### Шаг 3: Парсинг резюме с HH
- Система ищет резюме на HeadHunter по критериям
- Находит до 200 резюме
- Сохраняет их в базу данных

#### Шаг 4: ИИ-анализ
- Система извлекает ключевые концепции из запроса
- Проводит предварительную оценку всех резюме
- Выбирает топ-50 для глубокого анализа
- ИИ анализирует каждое резюме и выставляет оценки 1-10
- Генерирует краткое резюме кандидата
- Создает вопросы для собеседования

#### Шаг 5: Просмотр результатов
- Пользователь видит список кандидатов с оценками
- Может фильтровать по оценке, возрасту, зарплате
- Просматривает детали каждого кандидата

#### Шаг 6: Экспорт данных
- Пользователь экспортирует результаты в Excel
- Получает файл с данными всех кандидатов
