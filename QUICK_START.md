# 🚀 Быстрый запуск HH Resume Analyzer

## ✅ Что готово

- ✅ **Backend API** - полностью готов с авторизацией и поиском резюме
- ✅ **Frontend React App** - интегрирован с backend, система авторизации
- ✅ **MongoDB интеграция** - база данных для пользователей и поисков
- ✅ **AI анализ** - через Cloudflare Worker и Gemini AI

## 🎯 Запуск системы

### Вариант 1: Автоматический запуск
```bash
py start_system.py
```

### Вариант 2: Ручной запуск

#### 1. Запуск Backend
```bash
cd backend
py -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

#### 2. Запуск Frontend (в новом терминале)
```bash
cd frontend
npm start
```

## 📍 Доступ к системе

После запуска откройте в браузере:

- **🌐 Frontend приложение**: http://localhost:3000
- **📚 API документация**: http://localhost:8000/docs
- **📖 ReDoc**: http://localhost:8000/redoc

## 👤 Первый запуск

1. **Откройте** http://localhost:3000
2. **Нажмите "Войти"** в правом верхнем углу
3. **Выберите "Зарегистрироваться"**
4. **Заполните форму регистрации**:
   - Email: ваш email
   - Имя пользователя: любое
   - Пароль: минимум 8 символов
   - Полное имя: ваше имя
   - Компания: название компании
   - Должность: ваша должность
   - Роль: выберите подходящую

## 🎯 Использование системы

### 1. После входа в систему
- Увидите главную страницу с приветствием
- Будут показаны ваши последние поиски (если есть)

### 2. Создание поиска резюме
- Нажмите **"Начать новый поиск"**
- Введите **город** (например: Москва)
- Опишите **требования к кандидату** естественным языком
- Нажмите **"Найти кандидатов"**

### 3. Просмотр результатов
- Система найдет резюме с HeadHunter
- ИИ проанализирует каждое резюме
- Вы получите список с оценками и рекомендациями
- Можете экспортировать результаты в Excel

## 🔐 Роли пользователей

### Admin (Администратор)
- Полный доступ ко всем функциям
- Управление пользователями
- Доступ к панели администратора

### HR Manager (HR Менеджер)
- Создание поисков
- Просмотр всех результатов команды
- Экспорт данных

### HR Specialist (HR Специалист)
- Создание поисков
- Просмотр только своих результатов
- Экспорт своих данных

### Viewer (Наблюдатель)
- Только просмотр результатов
- Без возможности создания поисков

## 🛠️ Устранение проблем

### Frontend не запускается
```bash
cd frontend
npm install
npm start
```

### Backend не запускается
1. Проверьте, что MongoDB запущен
2. Проверьте файл `backend/.env`
3. Установите зависимости: `pip install -r backend/requirements.txt`

### Ошибки авторизации
1. Убедитесь, что backend запущен на порту 8000
2. Проверьте консоль браузера на ошибки
3. Очистите localStorage браузера

### Проблемы с поиском
1. Проверьте, что у вас есть API ключ Gemini в `backend/.env`
2. Убедитесь, что Cloudflare Worker доступен
3. Проверьте логи backend в консоли
