## 📋 Инструкции по развертыванию

### Быстрый старт

1. **Настройте переменные окружения:**
   ```bash
   cp backend/.env.example backend/.env
   # Отредактируйте backend/.env
   ```

2. **Сгенерируйте секретный ключ:**
   ```bash
   openssl rand -hex 32
   ```

3. **Запустите сервисы:**
   ```bash
   docker-compose up -d
   ```

4. **Проверьте здоровье:**
   ```bash
   curl http://localhost:8000/health
   ```

5. **Настройте резервное копирование:**
   ```bash
   # Добавьте в crontab
   0 2 * * * /path/to/scripts/backup_mongodb.sh
   ```

### Полная инструкция

См. `DEPLOYMENT_GUIDE.md` для детальных инструкций.
