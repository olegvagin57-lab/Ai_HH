### Frontend

---

## Инструкция по запуску

### 1. Backend (уже запущен)
```bash
cd backend
py -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 2. Frontend
```bash
cd frontend
npm install  # если еще не установлено
npm start
```

### 3. Откройте в браузере
http://localhost:3000
