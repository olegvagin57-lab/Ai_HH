## 🔧 API тестирование

### Backend API Endpoints

#### Health Check
- **GET** `/health` - проверка работоспособности
- **Ожидаемый код**: 200
- **Ожидаемый ответ**: `{"status": "healthy"}`

#### Authentication
- **POST** `/api/v1/auth/login` - вход в систему
- **POST** `/api/v1/auth/refresh` - обновление токена
- **GET** `/api/v1/auth/me` - текущий пользователь

#### Search
- **POST** `/api/v1/search/` - создание поиска
- **GET** `/api/v1/search/{id}` - получение поиска
- **GET** `/api/v1/search/{id}/status` - статус поиска
- **GET** `/api/v1/search/{id}/results` - результаты

#### Export
- **GET** `/api/v1/export/{id}/excel` - экспорт Excel
- **GET** `/api/v1/export/{id}/csv` - экспорт CSV
