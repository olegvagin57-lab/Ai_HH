# План реализации системы анализа резюме HH с ИИ

## Общее описание проекта
Система для автоматизированного подбора и фильтрации резюме с HeadHunter с использованием ИИ (Gemini) для глубокого анализа кандидатов.

## Основные компоненты
- **Backend**: FastAPI + Python
- **Frontend**: React.js
- **ИИ**: Google Gemini API
- **База данных**: PostgreSQL
- **Кеширование**: Redis
- **Парсинг**: parse_hh_data (временно до получения API ключа)

## Ключевые особенности реализации

### Система скоринга
1. **Предварительный скоринг**: Лемматизация + взвешенная оценка
2. **ИИ-скоринг**: Глубокий анализ топ-50 кандидатов
3. **Финальный рейтинг**: Комбинация обеих оценок

### Архитектура ИИ-анализа
- **HH-Concept-Extractor**: Легковесная модель для извлечения концепций
- **HH-Analyst**: Мощная модель для глубокого анализа резюме
- **AI-Detector**: Определение ИИ-сгенерированных резюме

### Экспорт и интеграция
- Экспорт в Excel с полной аналитикой
- API для интеграции с другими системами
- Система уведомлений для команды
