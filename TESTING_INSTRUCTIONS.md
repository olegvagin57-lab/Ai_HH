## 🎯 Что можно тестировать

### 1. Backend API (http://localhost:8001)

#### Основные endpoints:
- **GET** http://localhost:8001/ - главная страница API
- **GET** http://localhost:8001/health - проверка здоровья системы
- **GET** http://localhost:8001/api/v1/test - тест API
- **GET** http://localhost:8001/docs - Swagger документация

#### Как тестировать:
1. Откройте браузер
2. Перейдите на http://localhost:8001
3. Должны увидеть: `{"message": "HH Resume Analyzer Backend is running on port 8001!"}`
4. Откройте http://localhost:8001/docs для интерактивной документации
