# Интеграция парсера DarkDarW/hh.ru_pars

## Описание

Парсер из репозитория [DarkDarW/hh.ru_pars](https://github.com/DarkDarW/hh.ru_pars) интегрирован в проект как дополнительный вариант парсинга резюме с HeadHunter.

## Как это работает

Парсер использует BeautifulSoup для парсинга HTML-страниц hh.ru:
1. Ищет резюме по запросу через поисковую страницу
2. Извлекает ссылки на резюме
3. Парсит каждое резюме и извлекает данные (имя, возраст, зарплата, навыки и т.д.)

## Приоритет использования

Парсер DarkDarW используется как **приоритет 5** (последний в списке):
1. Search Cards Parser (наиболее надежный)
2. Full Page Parser
3. Kate Parser
4. parse_hh_data Parser
5. **DarkDarW Parser** ← новый

## Установка

Парсер уже клонирован в папку `hh.ru_pars/` в корне проекта.

### Зависимости

Добавлены в `requirements.txt`:
- `beautifulsoup4==4.12.2`
- `lxml==4.9.3`

### Автоматическое определение

Парсер автоматически определяется при запуске, если:
- Папка `hh.ru_pars/` существует в корне проекта
- Файл `hh.ru_pars/resume_parser.py` доступен

## Использование

Парсер используется автоматически, если:
- Нет API ключей HeadHunter
- Другие парсеры недоступны или не работают

## Особенности

- **Задержки**: Добавлены задержки между запросами (0.3-0.5 сек) для избежания блокировки
- **User-Agent**: Используется реалистичный User-Agent
- **Обработка ошибок**: Graceful handling ошибок парсинга отдельных резюме
- **Формат данных**: Автоматическое преобразование в формат, ожидаемый приложением

## Ограничения

1. **Скорость**: Парсинг медленнее, чем через API (нужно загружать и парсить HTML)
2. **Надежность**: Может сломаться при изменении структуры HTML на hh.ru
3. **Данные**: Извлекаются только базовые данные (имя, возраст, зарплата, навыки)
4. **Юридические**: Использование парсера может нарушать условия использования hh.ru

## Рекомендации

Для production рекомендуется:
1. Использовать официальное API HeadHunter (требует регистрации на dev.hh.ru)
2. Или использовать более надежные парсеры (search_cards, full_page)

## Тестирование

Для проверки работы парсера:
1. Убедитесь, что папка `hh.ru_pars/` существует
2. Запустите проект
3. Выполните поиск резюме через API
4. Проверьте логи - должен использоваться DarkDarW parser

## Примечания

- Оригинальный парсер содержит ошибку: `import ran` должно быть `import random` (исправлено в нашей интеграции)
- Парсер не использует оригинальный код напрямую, а реализует аналогичную логику с улучшениями
