# 👥 Создание тестовых пользователей
## Инструкция для администратора

### 🎯 Цель
После удаления регистрации пользователей, администратор должен создавать учетные записи вручную.

---

## 🔧 Способы создания пользователей

### Вариант 1: Через API (рекомендуется)

#### 1. Создание администратора (первый пользователь)
```bash
curl -X POST "http://localhost:8000/api/v1/auth/register" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "admin@company.com",
    "username": "admin",
    "password": "SecureAdmin123!",
    "full_name": "Системный Администратор",
    "company_name": "Ваша Компания",
    "position": "Системный администратор",
    "role_names": ["admin"]
  }'
```

#### 2. Создание HR менеджера
```bash
curl -X POST "http://localhost:8000/api/v1/users/" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -d '{
    "email": "hr.manager@company.com",
    "username": "hr_manager",
    "password": "HRManager123!",
    "full_name": "HR Менеджер",
    "company_name": "Ваша Компания",
    "position": "HR Менеджер",
    "role_names": ["hr_manager"]
  }'
```

#### 3. Создание HR специалиста
```bash
curl -X POST "http://localhost:8000/api/v1/users/" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -d '{
    "email": "hr.specialist@company.com",
    "username": "hr_specialist",
    "password": "HRSpec123!",
    "full_name": "HR Специалист",
    "company_name": "Ваша Компания",
    "position": "HR Специалист",
    "role_names": ["hr_specialist"]
  }'
```

#### 4. Создание наблюдателя
```bash
curl -X POST "http://localhost:8000/api/v1/users/" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -d '{
    "email": "viewer@company.com",
    "username": "viewer",
    "password": "Viewer123!",
    "full_name": "Наблюдатель",
    "company_name": "Ваша Компания",
    "position": "Аналитик",
    "role_names": ["viewer"]
  }'
```

### Вариант 2: Через админ панель (после авторизации)

1. Войдите как администратор
2. Перейдите в "Админ панель"
3. Откройте "Управление пользователями"
4. Нажмите "Добавить пользователя"
5. Заполните форму и сохраните

## 🔒 Политика паролей

### Требования к паролям:
- Минимум 8 символов
- Содержит буквы и цифры
- Рекомендуется использовать специальные символы
- Уникальный для каждого пользователя

### Примеры безопасных паролей:
- `SecureAdmin123!`
- `HRManager2024#`
- `Specialist$789`
- `ViewerPass456&`

## 🧪 Тестирование ролей

### Тест прав доступа:

#### Администратор должен иметь доступ к:
- ✅ Создание поисков резюме
- ✅ Просмотр всех результатов
- ✅ Управление пользователями
- ✅ Панель администратора
- ✅ Экспорт данных

#### HR Менеджер должен иметь доступ к:
- ✅ Создание поисков резюме
- ✅ Просмотр результатов команды
- ✅ Экспорт данных
- ❌ Управление пользователями

#### HR Специалист должен иметь доступ к:
- ✅ Создание поисков резюме
- ✅ Просмотр только своих результатов
- ✅ Экспорт своих данных
- ❌ Управление пользователями

#### Наблюдатель должен иметь доступ к:
- ✅ Просмотр результатов
- ❌ Создание поисков
- ❌ Управление пользователями
- ❌ Экспорт данных
