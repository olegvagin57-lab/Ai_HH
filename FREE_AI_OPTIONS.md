# Бесплатные варианты для семантического анализа резюме

## 1. Google Gemini API (текущий вариант)

### Преимущества:
- ✅ Бесплатная квота: 60 запросов/минуту, 1500 запросов/день
- ✅ Хорошее качество анализа
- ✅ Поддержка русского языка
- ✅ Уже интегрирован в проект

### Недостатки:
- ❌ Квота может закончиться
- ❌ Нужен API ключ

### Статус:
- Текущий Worker использует Gemini API
- При исчерпании квоты система переключается на fallback

---

## 2. Hugging Face Inference API (БЕСПЛАТНО)

### Преимущества:
- ✅ Полностью бесплатно (до 30,000 запросов/месяц)
- ✅ Много моделей на выбор
- ✅ Не требует кредитной карты
- ✅ Хорошее качество для анализа текста

### Популярные модели:
- `mistralai/Mistral-7B-Instruct-v0.2` - отличная для анализа
- `meta-llama/Llama-2-7b-chat-hf` - хорошая для русского
- `microsoft/DialoGPT-medium` - для диалогов
- `google/flan-t5-large` - для задач анализа

### Интеграция:
Нужно добавить клиент для Hugging Face API в проект.

---

## 3. OpenAI API (ограниченная бесплатная квота)

- ✅ Отличное качество (GPT-3.5-turbo)
- ✅ Хорошая поддержка русского языка

### Недостатки:

- ❌ Квота ограничена

## Рекомендации

### Для начала (быстро):
1. **Продолжить с Gemini** - уже работает, просто нужно восстановить квоту
2. **Использовать fallback** - уже работает, базовое качество

### Для лучшего качества (бесплатно):
1. **Hugging Face Inference API** - лучший бесплатный вариант
2. **Ollama локально** - если есть сервер, полностью бесплатно

### Для production:
1. **Gemini API** - хороший баланс цена/качество
2. **OpenAI GPT-3.5-turbo** - лучшее качество, но платно
