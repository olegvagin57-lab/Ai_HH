# 🎯 HH Resume Analyzer - Демо инструкции

## 🚀 Быстрый запуск демо

### 1. Запуск backend сервера
```bash
python run_simple.py
```

Сервер запустится на http://localhost:8000

### 2. Тестирование API

#### Через браузер:
- Откройте http://localhost:8000/docs
- Интерактивная документация Swagger

#### Через curl/Postman:

**Создать поиск:**
```bash
curl -X POST "http://localhost:8000/api/search/" \
  -H "Content-Type: application/json" \
  -d '{
    "city": "Москва",
    "query": "Нужен активный менеджер по продажам спецодежды B2B"
  }'
```

**Получить статус поиска:**
```bash
curl "http://localhost:8000/api/search/{search_id}/status"
```

**Получить результаты:**
```bash
curl "http://localhost:8000/api/search/{search_id}/resumes"
```

## 🎭 Что показывает демо

### Функциональность:
- ✅ Создание поиска резюме
- ✅ Фоновая обработка (имитация)
- ✅ Статусы: pending → processing → completed
- ✅ Mock данные с реалистичными резюме
- ✅ ИИ-анализ (имитация)
- ✅ Оценки от 1 до 10
- ✅ Вопросы для собеседования
- ✅ Детектор ИИ-резюме

### Mock данные включают:
- 10 кандидатов с разными профилями
- Реалистичные имена и должности
- Оценки от 10 до 1 (по убыванию)
- ИИ-саммари для каждого кандидата
- Готовые вопросы для собеседования
- Пометки об ИИ-сгенерированных резюме

## 🔧 Для полной функциональности

### Необходимо:
1. **HH.ru API ключи**:
   - Регистрация на https://dev.hh.ru/
   - Client ID и Client Secret

2. **Gemini API ключ**:
   - Работает в поддерживаемых регионах
   - Альтернатива: OpenAI API

3. **Полная среда**:
   - Docker + Docker Compose
   - Или полная установка Python зависимостей
   - PostgreSQL + Redis
