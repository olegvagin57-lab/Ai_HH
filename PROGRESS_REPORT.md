### 🚀 Демо версия включает:

#### Полный цикл работы:
1. **Создание поиска** - ввод города и описания вакансии
2. **Фоновая обработка** - имитация поиска и анализа
3. **Прогресс в реальном времени** - статусы и индикаторы
4. **Результаты с ИИ-анализом** - 10 mock резюме с оценками
5. **Статистика и фильтры** - анализ результатов
6. **Экспорт данных** - скачивание в CSV формате

#### Mock данные включают:
- 10 реалистичных резюме кандидатов
- Оценки от 10 до 1 (по убыванию качества)
- ИИ-саммари для каждого кандидата
- 3 вопроса для собеседования на каждого
- Пометки об ИИ-сгенерированных резюме (каждое 4-е)
- Разные зарплатные ожидания и возраста

### 🔧 Для полной функциональности нужно:

1. **API ключи**:
   - HH.ru Client ID и Client Secret
   - Gemini API ключ (в поддерживаемом регионе)

2. **Полная среда**:
   - Docker для запуска всех сервисов
   - PostgreSQL + Redis
   - Или ручная установка всех зависимостей
